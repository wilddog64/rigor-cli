## Context

`_curl` calls `_run_command --quiet` without `--soft`. When curl returns a non-zero exit
code (port 9222 not yet listening), `_run_command` calls `exit 1` — killing the entire
script on the first poll iteration. The retry loop never retries; the script just dies.

`_curl` must NOT be used as a boolean probe in an `if` statement. The correct pattern is
`_run_command --soft -- curl ...` which returns the exit code without calling `exit 1`.

## Change

In `_antigravity_browser_ready`, replace:

```bash
      if _command_exist curl && _curl -sf http://localhost:9222/json >/dev/null 2>&1; then
```

With:

```bash
      if _command_exist curl && _run_command --soft -- curl -sf http://localhost:9222/json >/dev/null 2>&1; then
```

That is the only change required.
