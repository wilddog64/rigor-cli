# Spec: _agent_audit Tab Indentation Check (v0.3.8)

## Goal

Add a tab indentation check to `_agent_audit` so that any staged `.sh` file containing
tab-indented lines fails the pre-commit hook. All shell scripts must use 2-space indentation.
The check scans the full staged file content — not just the diff — so pre-existing tabs are
caught the first time a file is committed after this check ships.

## Implementation

### `scripts/lib/agent_rigor.sh`

Add the following block inside `_agent_audit`, after the bare-sudo check block
(after line `fi` that closes the bare-sudo loop) and before the `staged_diff` credential check.

The block reuses the existing `changed_sh` variable (already populated above it):

```bash
   if [[ -n "$changed_sh" ]]; then
      local file
      for file in $changed_sh; do
         [[ -f "$file" ]] || continue
         local tab_lines
         tab_lines=$(git show :"$file" 2>/dev/null | grep -n $'^\t' || true)
         if [[ -n "$tab_lines" ]]; then
            _warn "Agent audit: tab indentation in $file — use 2-space indentation:"
            _warn "$tab_lines"
            status=1
         fi
      done
   fi
```

**Insertion point** — exact surrounding context to locate where to insert:

```bash
         if [[ -n "$bare_sudo" ]]; then
            _warn "Agent audit: bare sudo call in $file (use _run_command --prefer-sudo):"
            _warn "$bare_sudo"
            status=1
         fi
      done
   fi

   local staged_diff
```

Insert the new block between the closing `fi` of the bare-sudo loop and `local staged_diff`.

### `scripts/tests/lib/agent_rigor.bats`

Add two tests before the existing `_agent_audit ignores unstaged .sh changes` test:

```bash
@test "_agent_audit flags tab indentation in staged .sh file" {
  mkdir -p scripts
  printf 'function tabbed() {\n\techo "tab"\n}\n' > scripts/tabbed.sh
  git add scripts/tabbed.sh
  run _agent_audit
  [ "$status" -ne 0 ]
  [[ "$output" == *"tab indentation"* ]]
}

@test "_agent_audit passes with 2-space indentation" {
  mkdir -p scripts
  printf 'function spaced() {\n  echo "spaces"\n}\n' > scripts/spaced.sh
  git add scripts/spaced.sh
  run _agent_audit
  [ "$status" -eq 0 ]
}
```
