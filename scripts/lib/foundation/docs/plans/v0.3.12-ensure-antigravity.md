# v0.3.12 — `_ensure_antigravity_ide` + MCP Config Helpers

## Context

Antigravity is a Google agent-first IDE (VS Code fork) that ships with a browser automation
surface. When Antigravity runs with `--remote-debugging-port=9222`, a Playwright MCP server
can connect to the existing authenticated browser session — enabling agents to interact with
authenticated web UIs (GitHub Copilot coding agent, ACG sandbox) without session isolation.

This spec adds three helpers to `scripts/lib/system.sh`:

| Function | Purpose |
|---|---|
| `_ensure_antigravity_ide` | Install Antigravity via brew (macOS) / apt (Debian) / dnf (RedHat) |
| `_ensure_antigravity_mcp_playwright` | Inject Playwright MCP entry into Antigravity's `mcp_config.json` (idempotent) |
| `_antigravity_browser_ready` | Verify port 9222 is listening; retry with configurable timeout |

Private helper: `_antigravity_mcp_config_path` — locate `mcp_config.json` on disk.

## Implementation

### `_antigravity_mcp_config_path` (private)

Locate Antigravity's `mcp_config.json`. Returns the path via stdout.

```bash
function _antigravity_mcp_config_path() {
   if _is_mac; then
      printf '%s\n' "${HOME}/Library/Application Support/Antigravity/mcp_config.json"
      return 0
   fi
   # Linux: XDG_CONFIG_HOME or ~/.config
   local config_dir="${XDG_CONFIG_HOME:-${HOME}/.config}"
   printf '%s\n' "${config_dir}/Antigravity/mcp_config.json"
}
```

### `_ensure_antigravity_ide`

Install the Antigravity IDE. Follows the same pattern as `_ensure_node`.

```bash
function _ensure_antigravity_ide() {
   if _command_exist antigravity; then
      return 0
   fi

   if _is_mac && _command_exist brew; then
      _run_command -- brew install --cask antigravity
      if _command_exist antigravity; then
         return 0
      fi
   fi

   if _is_debian_family && _command_exist apt-get && _sudo_available; then
      _run_command --prefer-sudo -- apt-get install -y antigravity
      if _command_exist antigravity; then
         return 0
      fi
   fi

   if _is_redhat_family && _command_exist dnf && _sudo_available; then
      _run_command --prefer-sudo -- dnf install -y antigravity
      if _command_exist antigravity; then
         return 0
      fi
   fi

   _err "Cannot install Antigravity IDE: no supported package manager found or install failed"
}
```

### `_ensure_antigravity_mcp_playwright`

Inject the Playwright MCP server entry into Antigravity's `mcp_config.json`. Idempotent —
no-op if the `playwright` key already exists. Requires `jq`.

```bash
function _ensure_antigravity_mcp_playwright() {
   if ! _command_exist jq; then
      _err "_ensure_antigravity_mcp_playwright requires jq"
   fi

   local config_path
   config_path="$(_antigravity_mcp_config_path)"

   # Create config dir + file if missing
   local config_dir
   config_dir="$(dirname "$config_path")"
   if [[ ! -d "$config_dir" ]]; then
      mkdir -p "$config_dir"
   fi
   if [[ ! -f "$config_path" ]]; then
      printf '{"mcpServers":{}}\n' > "$config_path"
   fi

   # No-op if playwright entry already present
   if jq -e '.mcpServers.playwright' "$config_path" >/dev/null 2>&1; then
      return 0
   fi

   local tmp
   tmp="$(mktemp)"
   jq '.mcpServers.playwright = {"command":"npx","args":["-y","@playwright/mcp@latest"]}' \
      "$config_path" > "$tmp" && mv "$tmp" "$config_path"
}
```

### `_antigravity_browser_ready`

Check that port 9222 is listening (Antigravity running with remote debugging enabled).
Retries every 2 seconds up to `[timeout_seconds]` (default: 10).

```bash
function _antigravity_browser_ready() {
   local timeout="${1:-10}"
   local elapsed=0

   while [[ "$elapsed" -lt "$timeout" ]]; do
      if curl -sf http://localhost:9222/json >/dev/null 2>&1; then
         return 0
      fi
      sleep 2
      elapsed=$(( elapsed + 2 ))
   done

   _err "Antigravity browser not ready on port 9222 after ${timeout}s — launch Antigravity with --remote-debugging-port=9222"
}
```
